## FunkyLibrary, library for hytale

This is a library for all FunkyDev proyects and maybe thirdParty's in the future

### Features
- FunkyAssetReloadEvent dispatched on asset modified and waiting to reload
- FunkyAssetReloadedEvent dispatched on asset reloaded
